# ILPA PortCo Template — Dropdown Options Reference

Source: ILPA Portfolio Company Metrics Template v1.0 (Tab 12: Dropdown Options)

---

## Position Status
| Value | Definition |
|-------|-----------|
| Active, No Partial Exits | Position has not realized any invested capital/gains; no write-offs |
| Active, Partially Exited | Position has realized some capital/gains and/or partial write-off; Reported Value or Current Cost > 0 |
| Fully Exited, No Escrow | Position completely realized; no escrow pending; Reported Value and Current Cost = 0 |
| Fully Exited, Escrow Pending | Completely realized but portion of proceeds in escrow account |
| Fully Exited, Subsequently Closed | Position was fully realized after Template Reporting Date but before current Template |
| Pending, Subsequently Closed | Initial investment completed after Template Reporting Date but before current Template |

## Position Type
| Value | Definition |
|-------|-----------|
| Equity - Publicly Traded | Ownership in company; tradable on public exchange |
| Equity - Private | Ownership in company; NOT tradable on public exchange |
| Debt | Loan to company; either Originated or Purchased |
| Derivative | Contract between parties based on underlying asset(s) |

## Investment Period Status
| Value | Definition |
|-------|-----------|
| Evaluating New Investments | Fund is in investment period; can invest in new companies |
| Follow-On Investments Only | Outside investment period; can only do follow-ons |
| No New or Follow-On Investments | Outside investment period; no new investments possible |

## Investment Strategy
| Value | Definition |
|-------|-----------|
| Buyout | Control investment in mid/late-stage businesses with stable cash flows |
| Growth | Minority investment in profitable early life-cycle businesses needing expansion capital |
| Venture | Minority investment in early life-cycle businesses needing starter capital |
| Growth/VC for Control | Controlling interest in early life-cycle business |
| Credit Related | Debt/credit-focused investment strategy |
| Direct Lending | Loan to financially stable businesses |
| Distressed | Loan to financially troubled company for above-average yield or control |
| Mezzanine | Hybrid debt/equity with conversion rights |
| Fund of Funds | LP interest in a closed-ended fund at/before final closing |
| Secondary | LP interest acquired from another investor after target fund's final closing |
| Real Estate | Equity or debt investment in property or property-securitized assets |
| Natural Resources | Precious metals, commodities, agricultural land, oil/gas, similar real assets |
| Infrastructure | Equity or debt in physical structures/networks providing essential services |

## Deal Role
| Value | Definition |
|-------|-----------|
| Lead | GP led the negotiation of the transaction |
| Co-Lead | GP took key role in negotiation alongside other active participants |
| Minority Co-investor | GP participated on terms negotiated by a third party |

## Purchase Process
| Value | Definition |
|-------|-----------|
| Public | Includes Public-to-Private and Open Market transactions |
| Full Auction | Broad auction with multiple participants reviewing and bidding |
| Partial Auction | Narrow auction with limited number of participants (e.g., 2-3) |
| Broker | Non-formal auction; broker identifies one or multiple possible purchasers |
| Proprietary Sourcing | Non-brokered or auctioned private process |

## Purchase Type
| Value | Definition |
|-------|-----------|
| Public to Private | Publicly listed business is delisted |
| Corporate Carve-Out | Division or part of larger business is purchased |
| Management Buyout | Fund partnered with existing management to acquire business |
| Founder Succession | Previously founder-owned; owner selling majority stake |
| Financial Sponsor | Secondary transaction from another PE/investment firm |
| State/Government | Counterparty is state or government (privatization) |
| Debt Purchase | Purchase of debt; typically to convert to equity or benefit from recovery |
| Capital Increase | Provision of start-up capital, growth capital, or equity recapitalization |
| ESOP | Financing for acquisition of business majority-owned by employee stock option plan |

## Final Exit Type
| Value | Definition |
|-------|-----------|
| Strategic Buyer | Sold to business in same or complementary industry |
| Financial Buyer | Sold to investor specializing in buying businesses with growth potential |
| IPO | Company listed on public exchange; fund's shares sold |
| Full Write-Off | Position deemed permanently impaired |
| Partial Write-Off + Partial Sale | Portion permanently impaired; remaining realized via sale |
| Recapitalization - Minority Investment | Sale of minority stake without change in control |
| Reached Maturity Date | Principal returned at pre-specified date (debt) |
| Early Redemption | Principal returned before maturity date (debt) |

## Debt Status
| Value | Definition |
|-------|-----------|
| Performing | Debt not in or near default; interest and principal payments current |
| Non-Performing | Debt in default or close to default; payments not being made |
| No Debt | Company has no debt |

## Covenant Status
| Value | Definition |
|-------|-----------|
| Compliant | Debt performing; no covenant breach expected |
| Breach | Covenants breached; negotiations with creditors not yet started |
| Expected Breach | Company is expected to breach covenants |
| Waived | Covenants breached but creditors elected to temporarily forgo enforcement |
| No Covenant | Debt facility has no covenants |
| Renegotiating (breach) | Company amending covenants due to breach |
| Renegotiating (no breach) | Company proactively amending covenants (no breach) |

## Valuation Methodology
| Value | Definition |
|-------|-----------|
| Multiple of Revenue | Appropriate multiple × company Revenue during reference period |
| Multiple of Recurring Revenue | Appropriate multiple × company Recurring Revenue |
| Multiple of Earnings (P/E) | Appropriate multiple × Earnings per Share |
| Multiple of EBIT | Appropriate multiple × Earnings Before Interest and Taxes |
| Multiple of EBITA | Appropriate multiple × Earnings Before Interest, Taxes, and Amortization |
| Multiple of EBITDA | Appropriate multiple × EBITDA |
| Multiple of EBITDA, less CapEx | Appropriate multiple × (EBITDA - CapEx) |
| Multiple of EBITDA, less Minority Interests | Appropriate multiple × (EBITDA - Minority Interests) |
| Industry Valuation Benchmark | Industry-specific benchmarks (e.g., price per subscriber) |
| Available Market Prices - Quoted | Bid/ask price from active market |
| Available Market Prices - Quoted, less discount | Quoted price with discount for contractual restrictions |
| Observable Prices - Third-Party Sources | Pricing services, broker/dealer quotes |
| Discounted Cash Flows or Earnings (of Investee Company) | DCF of business cash flows |
| Discounted Cash Flows (from an Investment) | DCF of investment cash flows |
| Net Assets | Value derived from net asset value |

## Financial Reference Period
| Value | Definition |
|-------|-----------|
| Actuals TTM | Actual values for last 12 months ending on Analysis Date |
| Forecasted TTM | Estimated values for 12-month period ending at future Analysis Date |
| Adjusted TTM | Actual values adjusted for non-recurring items, last 12 months |
| Pro-forma TTM | Actual TTM adjusted to reflect future transaction or catalyst |
| As At | Current balance as of Analysis Date (not TTM activity) |
| n/a | Methodology does not require a Financial Reference Period (e.g., DCF) |

## Transaction Type (Returns Template)
| Value | Definition |
|-------|-----------|
| Investment | Fund's investment into position; expressed as negative value |
| Realization | Proceeds received (sales, dividends, interest); expressed as positive value |
| Reported Value | Fair market value of unrealized position as of reporting date; expressed as positive value |
